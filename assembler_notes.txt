Start:
In
Store .Buffer_Index
Swap
Load 0
Swap
Subtract
Branch to Next if 0
Load Buffer_Index
Swap
Load 1
Add
Store Buffer_Index
Branch to Start

Next:
Check A, .Ptr
Branch to A if True
Check S, .Ptr
Branch to S if True
Check L, .Ptr
Branch to L if True
Check B, .Ptr
Branch to B if True
Check I, .Ptr
Branch to I if True
Check O, .Ptr
Branch to O if True
Load .Ptr
Branch to End if 0
Add 1 to Ptr
Branch to Next
